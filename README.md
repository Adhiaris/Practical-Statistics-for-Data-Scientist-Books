# Practical Statistics for Data Scientists

A hands-on companion to *Practical Statistics for Data Scientists* (Bruce, Bruce & Gedeck), covering core statistical concepts and machine learning methods with Python code and real-world datasets.

## About This Book

*Practical Statistics for Data Scientists* bridges the gap between classical statistics and modern data science. It covers the statistical foundations that every data scientist needs — from exploratory analysis and sampling distributions to regression, classification, and unsupervised learning — with a focus on practical application over mathematical formalism.

## Who Should Read This Book

This book is designed for practitioners who want to apply statistical thinking to real data problems. It assumes:

- Basic familiarity with Python
- Some exposure to data analysis or machine learning
- No advanced statistics background required

Topics such as probability distributions, hypothesis testing, and linear algebra are explained from the ground up as needed.

## Roadmap

| Chapter | Topic | Description |
|---------|-------|-------------|
| 1 | Exploratory Data Analysis | Measures of location, variability, distributions, and visualization techniques |
| 2 | Data and Sampling Distributions | Sampling distributions, the bootstrap, confidence intervals, and common probability distributions |
| 3 | Statistical Experiments and Significance Testing | A/B testing, permutation tests, t-tests, ANOVA, chi-square tests, and statistical power |
| 4 | Regression and Prediction | Linear regression, model selection, regularization (Lasso), polynomial terms, and regression diagnostics |
| 5 | Classification | Naive Bayes, discriminant analysis, logistic regression, ROC/AUC, and strategies for imbalanced data |
| 6 | Statistical Machine Learning | K-Nearest Neighbors, decision trees, random forests, and XGBoost with hyperparameter tuning |
| 7 | Unsupervised Learning | PCA, correspondence analysis, K-Means, hierarchical clustering, and Gaussian Mixture Models |


## Prerequisites

- Basic Python programming experience
- Familiarity with pandas and matplotlib is helpful but not required
